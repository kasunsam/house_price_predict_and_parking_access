House Price Prediction & Parking Access Analysis

Developed a machine learning project in Python for predicting house prices and analyzing parking accessibility using geospatial and real estate datasets.

Implemented predictive analytics, data preprocessing, feature engineering, and visualization techniques to identify property value trends and location-based insights.

Utilized Python libraries including Pandas, NumPy, Scikit-learn, Matplotlib, and GIS-based mapping tools for data analysis and model development.

Built interactive analytical models and mapping solutions to support data-driven urban planning and real estate decision-making.
